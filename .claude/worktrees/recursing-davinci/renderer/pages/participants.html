Pro Tip: Use Dynamic Keywords!
Type { to see available keywords that will be replaced automatically:
• {number} = Race number (e.g., "51")
• {name} = Person name
• {team} = Team name
• {category} = Category
• {tag} = Custom tag

Example: {number}-{name} → "51-A. Pier Guidi"
